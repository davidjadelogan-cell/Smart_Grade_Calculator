# Smart_Grade_Calculator
Description

This Python program works as a simple student grade calculator.
It collects a student's:
- Test score
- Assignment score
- Exam score

Then program the calculates the:
- Total score
- Average score

Then checks whether the student:
- Is eligible for an Award
- Passed
- Failed

# Features
- Accepts user input
- Performs arithmetic calculations
- Uses comparison operators
- Uses logical conditions
- Displays pass/fail results
- Checks eligibility for awards

# Requirements
Python 3 

# Concepts Used
- Variables
- Input Prompting
- Arithmetic Operations
- Conditional Statements (if, elif, else)
- Logical Operators
